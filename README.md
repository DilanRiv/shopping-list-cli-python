
# 🛒 Shopping List CLI / Aplicación de Listas de Compras CLI

A command-line application built with Python that allows users to create, view, edit, and delete shopping lists using text files for data storage.

Una aplicación de consola desarrollada en Python que permite crear, visualizar, editar y eliminar listas de compras utilizando archivos de texto para almacenar la información.

---

## ✨ Features / Características

### 🇬🇧 English

* Create new shopping lists.
* View all saved shopping lists.
* Edit existing lists by adding new items.
* Delete shopping lists.
* Prevent duplicate list names.
* Input validation for user selections.
* Automatic creation of the storage folder.
* Exit options inside submenus for better navigation.
* Lists are automatically sorted alphabetically.

### 🇪🇸 Español

* Crear nuevas listas de compras.
* Visualizar todas las listas guardadas.
* Editar listas existentes agregando nuevos productos.
* Eliminar listas de compras.
* Evitar nombres duplicados en las listas.
* Validación de entradas del usuario.
* Creación automática de la carpeta de almacenamiento.
* Opción de salir desde los submenús para mejorar la navegación.
* Las listas se ordenan automáticamente alfabéticamente.

## 🚀 Getting Started / Cómo Empezar

### 🇬🇧 English

1. Clone this repository:

```bash
git clone https://github.com/DilanRiv/shopping-list-cli-python.git
```

2. Navigate to the project directory:

```bash
cd shopping-list-cli-python
```

3. Run the application:

```bash
python todo_list.py
```

---

### 🇪🇸 Español
